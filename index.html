Sobre Mim

Sou um desenvolvedor e designer web especializado na criação de sites modernos, funcionais e visualmente atraentes. Combino criatividade e funcionalidade para transformar ideias em interfaces intuitivas, equilibrando design, usabilidade e código de qualidade.

Minha jornada na programação começou há 3 anos, ainda na faculdade, e desde então venho me dedicando ao aprimoramento contínuo no desenvolvimento web. Ao longo desse período, tenho aprofundado meus conhecimentos em Full Stack JavaScript, buscando aplicar boas práticas, código limpo e atenção aos detalhes para a criação de soluções eficientes e de qualidade.

Meus Projetos

DevCurrency

Aplicação desenvolvida para exibir informações de criptomoedas em tempo real, consumindo dados de uma API pública. O projeto foi construído com foco em boas práticas de organização, roteamento e manipulação de dados.

React.js TypeScript CSS

Dev_Reels

DEV REELS é uma plataforma de filmes desenvolvida a partir do consumo da API do TMDB. O projeto permite listar filmes, visualizar os detalhes de cada filme, salvar favoritos no localStorage do navegador e gerenciar essa lista, adicionando ou removendo conforme a preferência do usuário.

React.js localStorage CSS

DevLinks

DevLink é uma plataforma estilo Linktree. O sistema permite que cada usuário cadastre e personalize seus próprios links, além de gerenciar suas redes sociais de forma simples, prática e totalmente responsiva.

React.js TypeScript TailwindCSS Firebase

Shadow of Gotham

Sistema de controle de estoque que possui autenticação segura com login e JWT, proteção de rotas e funcionalidades completas de CRUD para usuários e recursos, garantindo gerenciamento eficiente e acesso controlado às áreas do sistema.

React.js Node.js Express MySQL JWT
